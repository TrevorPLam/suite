# Tooling Recommendations

| Category | Recommended Tools (2026) |
|----------|---------------------------|
| Version Control | GitHub / GitLab |
| CI/CD | GitHub Actions, GitLab CI, Buildkite |
| Monorepo Management | Turborepo, Nx, Bazel, pnpm workspaces |
| Pre-commit Hooks | `pre-commit` (Python), `lefthook` (Go), `husky` (Node) |
| Commit Lint | `commitlint`, `commitizen` |
| Code Quality | SonarQube, ESLint, Ruff, golangci-lint |
| Testing (Unit/Integration) | Jest, pytest, JUnit, Vitest |
| BDD | Cucumber, Behat, SpecFlow, Cypress (component) |
| Performance Testing | k6, Artillery, Locust |
| Accessibility Testing | axe-core, Pa11y, Playwright (with axe) |
| Security (SAST) | CodeQL, Semgrep, SonarQube |
| Security (Secrets) | Gitleaks, TruffleHog, detect-secrets |
| Supply Chain | Trivy, Snyk, Sigstore (`cosign`), SLSA framework |
| Dependency Updates | Renovate (preferred), Dependabot |
| Release Automation | `semantic-release`, Release Please, Changesets |
| Documentation | MkDocs, Docusaurus, Storybook, OpenAPI Generator |
| API Design | Stoplight, Swagger Editor, GraphQL Mesh |
| Design Tokens | Style Dictionary, Theo |
| Carbon Awareness | Carbon Aware SDK, Cloud Carbon Footprint |
| Chaos Engineering | Chaos Mesh, Gremlin, AWS FIS |
| Platform Engineering | Backstage, Humanitec, `platform` directory (DIY) |
| Observability | Grafana, Prometheus, Datadog, OpenTelemetry |
| Incident Management | PagerDuty, Opsgenie, incident.io |
| IaC | Terraform, Pulumi, OpenTofu |
| Database Migrations | Flyway, Alembic, golang-migrate, Prisma Migrate |
