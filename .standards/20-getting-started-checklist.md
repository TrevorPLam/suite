# Getting Started Checklist

For new repositories or teams starting with the Diamond Standard, implement these top‑5 highest‑impact practices first:

1. **Branch protection** on `main` (require PR, status checks, approvals)
2. **Linting & formatting** automated in CI and pre‑commit
3. **Conventional commits** with commitlint
4. **CI pipeline** that builds and runs unit tests on every push
5. **Automated dependency updates** with Renovate (security patches auto‑merge)

Once these are in place, progress through the [Adoption Roadmap](18-adoption-roadmap-maturity-model.md) towards Silver.
